Como usar

  1. Clique em Sortear elemento para sortear um elemento aleatório.
  2. O painel "Último sorteado" mostra o elemento atual; "Elementos sorteados" mostra o histórico.
  3. Clique em Gerar cartela para criar uma cartela 5×5 com elementos aleatórios.
  4. Você pode baixar a cartela como imagem com Baixar cartela (PNG).
